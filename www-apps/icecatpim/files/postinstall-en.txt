0. IcecatPIM has been installed into ${MY_INSTALLDIR}

1. You have to import sql :
 # mysql
 > CREATE DATABASE icecatpim DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci;
 > GRANT ALL PRIVILEGES ON icecatpim.* TO icecatpim@localhost
    IDENTIFIED BY 'password';
 > quit
 
 # mysql icecatpim < ${MY_INSTALLDIR}/sql/pim.sql

2. Set DB name, user and password :
    sed -e "s:*DBNAME*:icecatpim:g" -e "s:*DBU*:icecatpim:g" -e "s:*DBP*:password:g" -i ${MY_INSTALLDIR}/lib/atomcfg.pm

3. Configure Apache
    There is an example of vhost in ${MY_INSTALLDIR}/conf/pim_vhost.conf

4. Go go go ! Default user/pass : root/root
